# Dawson's configs (dotfiles)

## Setup
Right now I'm using [dotbot](https://github.com/anishathalye/dotbot).

This repo contains a submodule that will install dotbot automatically.
As long as you have Python 3.8+, `./install` will initialize the submodule and run dotbot to create the symlinks (see [install.conf.yaml](./install.conf.yaml)).
